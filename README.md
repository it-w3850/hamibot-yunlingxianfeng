在手机上安装完成Hamibot之后，在[脚本市场](https://hamibot.com/marketplace)搜索["云岭先锋"](https://hamibot.com/marketplace/fBLh2)并安装脚本

请确保手机满足以下条件
1. 打开无障碍服务权限、程序保持后台运行
2. 手机打开勿扰模式，防止突然的信息弹窗导致脚本的失败
